# Hexis-Room Planner

Αυτόνομη **AR εφαρμογή αποτύπωσης κάτοψης** (PWA) για κινητά με ARCore/WebXR.
Σκαρίφημα χώρου με την κάμερα — τοίχοι με πάχος, ανοίγματα (πόρτες, παράθυρα,
μπαλκονόπορτες, κενά) και εξαγωγή **DXF με layers** για ProgeCAD.

## Δυνατότητες

- **Κάτοψη / σκαρίφημα** — αποτύπωση τοίχων γωνία-γωνία με ρυθμιζόμενο πάχος (10–50 cm)
- **Ανοίγματα** — πόρτες, παράθυρα, μπαλκονόπορτες, κενά/διακοπές πάνω σε κάθε τοίχο
- **DXF export με χωριστά layers** — `TOICHOI`, `PORTES`, `PARATHYRA`,
  `MPALKONOPORTES`, `KENA`, `DIASTASEIS`
- **Επιπλέον μετρήσεις** — εμβαδόν δωματίου, απόσταση 2 σημείων, ύψος
- **Ακρίβεια** — πραγματικό WebXR hit-test στο σημείο στόχευσης + προθέρμανση
  tracking πριν τη μέτρηση
- **Βαθμονόμηση** — διόρθωση κλίμακας με γνωστή απόσταση
- **PWA** — εγκαθίσταται στην αρχική οθόνη, δουλεύει σε πλήρη οθόνη

## Απαιτήσεις

- Κινητό με **ARCore** (Android) και browser με υποστήριξη **WebXR** (Chrome)
- Σερβίρισμα μέσω **HTTPS** (το WebXR δεν τρέχει τοπικά/χωρίς https)

## Χρήση

1. Άνοιξε την εφαρμογή από κινητό (https)
2. Στρέψε αργά την κάμερα στο πάτωμα μέχρι να βρεθεί επιφάνεια (προθέρμανση)
3. Σύρε το δάχτυλο, σημάδεψε γωνίες τοίχων, πρόσθεσε ανοίγματα
4. Πάτα ✓ → δες την κάτοψη και κατέβασε το DXF

## Παραμετροποίηση

Όλες οι ρυθμίσεις ταυτότητας βρίσκονται στο `APP_CONFIG` στην αρχή του script:

```js
window.APP_CONFIG = {
  name:      'Hexis-Room Planner',
  shortName: 'RoomPlanner',
  version:   'v1.9 · floor-plan editor',
  domain:    '',            // π.χ. 'roomplanner.gr'
  accent:    '#E8772E',     // primary
  accent2:   '#3ddc97',     // secondary
  bg:        '#0A1730'      // background
};
```

## Τεχνικά

Single-file εφαρμογή (HTML + CSS + JS). Το three.js φορτώνεται από CDN με
fallback. Καμία τοπική εξάρτηση — αρκεί το ένα αρχείο.
